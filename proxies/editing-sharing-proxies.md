# Editing and Sharing Proxies

## Editing a proxy

1. Go to the **Proxy** section in the left sidebar
2. Click the pencil icon next to the proxy you want to edit
3. Modify the credentials, host, or port
4. Save

Changes apply immediately to all profiles using that proxy.

## Sharing proxies with team members

You can grant team members access to specific proxies in two ways:

### During profile transfer
When you transfer a profile to another user, you can include the proxy in the transfer. The receiving user gets access to the proxy along with the profile.

### From the Proxy section
1. Find the proxy in the **Proxy** section
2. Click the share icon
3. Select the team members to share with

Shared proxies appear in the receiving user's proxy list and can be used when creating or editing their profiles.

## Deleting a proxy

1. Go to the **Proxy** section
2. Select the proxy (or multiple proxies using checkboxes)
3. Click **Delete**

Deleting a proxy removes it from all profiles that were using it. Those profiles will show an empty proxy field.

## Proxy naming

Give proxies descriptive names to make them easier to manage at scale. For example: `US-Residential-1`, `DE-Mobile-Asocks`, `NodeMaven-Pool-1`. Set the name in the proxy settings when adding or editing.
