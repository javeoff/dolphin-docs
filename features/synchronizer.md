# Profile Synchronizer

The Profile Synchronizer lets you perform actions on one master profile and have all synchronized profiles mirror those actions in real time. Available on Starter and above.

## When to use it

- Bulk account farming (registrations, warm-ups)
- Parallel advertising campaign management
- Token management across many wallets
- Repetitive tasks that need to run on many accounts simultaneously

## How to start a synchronization session

1. In the Browser Profiles view, select the profiles you want to synchronize using checkboxes
2. Click the **Synchronizer** icon in the bulk operations bar that appears
3. Choose a master profile by toggling its slider
4. Click **Start synchronization**
5. Arrange all browser windows side-by-side at equal sizes
6. Perform your actions in the master profile — all other profiles mirror them
7. Click **Stop** when done and wait for the green confirmation checkmark

## Requirements for correct operation

- **Window sizes must match**: Windows of different sizes prevent action duplication. Resize all windows to identical dimensions before starting.
- **Same tab count**: All profiles should have the same number of open tabs.
- **Same extensions**: Profiles work best when they have identical extension configurations.

## Full-screen mode behavior

In full-screen mode, you can switch tabs but navigation clicks and typing are blocked. Use windowed mode for full synchronization functionality.

## Limitations

- Multiple profile groups cannot run simultaneously at specific sizes (planned for future updates)
- Some websites block simultaneous logins from the same IP address — use different proxies per profile
- Profile copies used in synchronization should be identical except for cookies and fingerprints

## Other profiles while synchronizing

Other profiles in your account can run independently without interrupting an active synchronization session.
