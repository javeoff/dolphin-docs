# Tags, Statuses, Notes, and Folders

Dolphin Anty provides several tools to organize and search through your browser profiles.

## Tags

Tags are custom labels you attach to profiles for grouping and filtering.

**Create a tag:**
1. Go to **Settings → Tags** or create one inline when editing a profile
2. Enter a tag name and choose a color
3. Assign the tag to profiles

Tags appear as colored chips in the profile list. Use the Filter panel to show only profiles with specific tags.

**Bulk tagging:**
Select multiple profiles with checkboxes and use the bulk operations bar to assign or remove tags from all selected profiles at once.

## Statuses

Statuses indicate the current state of a profile (active, paused, banned, in use, etc.). Create custom statuses with names and colors to match your workflow.

Use statuses combined with filters to quickly find profiles in a particular state.

## Notes

Notes are free-text fields attached to a profile. Use them to store login credentials reminders, account details, campaign names, or any other context.

Notes appear in the profile list and are searchable through the Filter panel's notes content filter.

## Folders

Folders group profiles into named collections, similar to directories.

**Create a folder:**
1. Click **New folder** in the left sidebar under Browser Profiles
2. Name the folder

**Add profiles to a folder:**
- Single profile: Open the 3-dot menu → **Add to folder**
- Multiple profiles: Select with checkboxes → bulk operations → **Add to folder**

**Folder sharing:**
Share folders with team members and assign specific permissions per user (view, edit, delete, launch, share, move). See [Teamwork](teamwork.md) for details.

## Filtering profiles

Open the **Filter** panel to search by:

| Filter | Options |
|--------|---------|
| User | Filter by the assigned user |
| Tags | Show profiles with specific tags |
| Status | Filter by status |
| Profile type | Facebook, Google, TikTok, Crypto, None |
| Proxy type | HTTP, SOCKS4, SOCKS5 |
| Notes | Text search within notes |
| Folder | Show profiles in a specific folder |

You can apply up to 5 filters at the same time.

## Pinning profiles

Pin important profiles to the top of the list via the 3-dot menu → **Pin on top**. Pinned profiles always appear first regardless of sort order.

## Launch history column

Add a **Launched** column to the profile list to see the most recent launch timestamp for each profile. Enable it from the column customization menu in the profile list header.
