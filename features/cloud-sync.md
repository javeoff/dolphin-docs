# Cloud Sync

Cloud Sync stores your browser profiles, cookies, and extensions in the cloud so you can access them from any device.

## How it works

When Cloud Sync is enabled for a profile, Dolphin Anty uploads the profile data (settings, cookies, extensions) to cloud storage. Any device logged into the same account can then download and run that profile.

## Enabling Cloud Sync for a profile

1. Open the 3-dot menu next to the profile
2. Click **Cloud Sync**
3. Toggle it on

You can also enable Cloud Sync during profile creation in the profile settings.

## Disabling Cloud Sync (local profiles)

On paid plans, you can disable Cloud Sync and store profiles locally only. This keeps your data entirely on your device.

To store a profile locally:
1. Open the 3-dot menu → **Cloud Sync**
2. Toggle it off

The profile data stays on your machine and is not uploaded to any server.

## Sync across devices

Multiple users can log into the same account from different devices. Opening the same profile on two devices simultaneously causes sync conflicts — always stop a profile on one device before opening it on another.

## Cookie sync

Cookies are synced as part of the profile data. If your cookies are being lost, common causes include:

- Power loss or network interruption during profile shutdown (session didn't close cleanly)
- Outdated User-Agent triggering cookie invalidation on some sites
- Antivirus interference during the sync process
- Free plan: cookies are stored locally only, not synced

## Extensions sync

Extensions installed through the Dolphin Anty Extensions menu sync alongside profile data on paid plans.
