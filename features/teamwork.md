# Teamwork

Dolphin Anty supports multi-user teams with role-based access control. Teamwork features are available on Base, Team, and Enterprise plans.

## User roles

| Role | Capabilities |
|------|-------------|
| **Admin** | Full access to all profiles, all users, all settings. Can create users, assign Teamleads, set passwords, and delete accounts. |
| **Teamlead** | Manages profiles belonging to assigned users. Can view, launch, and edit their profiles but cannot change passwords or remove users. |
| **User** | Creates and manages their own profiles. Can share profiles with teammates. Cannot manage other users. |

## Adding team members

1. Go to **Payment** in your personal area
2. Click **Upgrade plan** and purchase additional user slots
3. Create the new user in your team settings using an email that is not already registered in Dolphin Anty
4. Assign a role and configure permissions

Additional users cost:
- Base plan: $10/user
- Team plan: $20/user
- Enterprise plan: $25/user

## Sharing and transferring profiles

### Share a profile
Sharing gives another user access to the profile without removing it from your account. You can set specific permissions per share.

### Transfer a profile
Transferring moves the profile to another user's account. The profile disappears from your account after the transfer. Paid accounts can transfer profiles to free accounts; free accounts cannot transfer.

## Folder permissions

When you share a folder, you can grant granular permissions:

- View folder and its profiles
- Modify folder settings
- Delete the folder
- Launch profiles
- Edit profiles
- Share profiles with others
- Delete profiles
- Move profiles in and out of the folder

## Profile creation limits

Admins can restrict how many profiles each user can create. Enable this via the user settings panel with the **Allow creating browser profiles** toggle.

## Viewing team activity

Use **Filter → Advanced Search** to view profiles belonging to specific users. The **History** feature on each profile shows when it was launched and by whom.

## Removing users

Before deleting a user, decide what happens to their profiles:

- Toggle **Transfer profiles to my account** to move their profiles to your account
- Leave it off to permanently delete their profiles along with the user

Local profiles (not cloud-synced) cannot be transferred and will be deleted regardless.

## Filtering by user

In the Browser Profiles view, use the **Filter** panel to filter profiles by the user they belong to or are assigned to.
