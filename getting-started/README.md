# Quick Start

Get Dolphin Anty running and create your first browser profile in under 10 minutes.

## Step 1: Create an account

1. Go to [dolphin-anty.com](https://dolphin-anty.com) and click **Create account**
2. Enter your email address and a secure password
3. Provide your Telegram handle (used for support and verification)
4. Optionally enter a promo code
5. Accept the Terms of Use and Privacy Policy, then submit

After registration, your personal area opens automatically. You'll be prompted to enable two-factor authentication — this is optional but recommended.

## Step 2: Download and install

Download the app for your operating system:

- **Windows**: [dolphin-anty.com/download/windows](https://dolphin-anty.com/download/windows/) (64-bit and 32-bit)
- **macOS**: [dolphin-anty.com/download/macos](https://dolphin-anty.com/download/macos/) (supports M1/M2/M3)
- **Linux**: See the [Linux installation guide](installation-linux.md)

Run the installer and launch Dolphin Anty. The app downloads browser components on first launch — wait for this to complete before logging in.

## Step 3: Log in

Use the email and password you created during registration.

## Step 4: Create your first profile

1. Go to **Browser Profiles** in the left sidebar
2. Click the **+** button in the top-right corner
3. Enter a profile name
4. (Optional) Add a proxy — see [How to add a proxy](../proxies/add-proxy.md)
5. (Optional) Paste cookies
6. Select a profile type: Facebook, Google, TikTok, Crypto, or None
7. Click **New fingerprint** if you want to review the generated fingerprint
8. Click **Create profile**

The default fingerprint settings are optimal for most users. Avoid changing them unless you have a specific reason.

## Step 5: Start the profile

Click **Start** next to your profile. A browser window opens with the isolated environment. Close the browser window or click **Stop** in the app to end the session.
