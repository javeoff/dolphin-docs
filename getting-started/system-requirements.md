# System Requirements

## Minimum requirements

| Component | Requirement |
|-----------|-------------|
| OS | Windows 10+, macOS 10.14+, Ubuntu 18.04+ (or equivalent) |
| CPU | Dual-core processor, 2.0 GHz |
| RAM | 4 GB |
| Storage | 10–15 GB free space on the system drive |
| Internet | Stable broadband connection |

## Recommended

| Component | Recommendation |
|-----------|----------------|
| RAM | 8 GB or more |
| Storage | 20+ GB free on SSD |
| Internet | Low-latency connection for faster profile syncing |

## macOS Apple Silicon

Dolphin Anty natively supports M1, M2, and M3 processors. Download the macOS build from the download page — no Rosetta required.

## Storage note

Keep at least 10–15 GB free on your C: drive (Windows) or system partition (macOS/Linux). Insufficient disk space is one of the most common causes of profile corruption errors (`Error 409: Datadir damaged`).

## Antivirus compatibility

Some antivirus software can interfere with profile launches. Known conflicting programs include:

- RAV Endpoint Protection
- Bitdefender
- Avast

If profiles fail to start, temporarily disable your antivirus to verify whether it's causing the issue. Add Dolphin Anty to your antivirus exclusions list to resolve the conflict.
