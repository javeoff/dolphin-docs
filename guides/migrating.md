# Moving to Dolphin Anty from Another Browser

If you're switching from another antidetect browser, here's how to bring your accounts with you.

## What can be transferred

There is no direct automated import from other antidetect browsers. You can manually transfer:

- **Cookies** — export from the old browser and import into Dolphin Anty profiles
- **User-Agent strings** — copy the User-Agent from the old profile when creating the new one

Profile settings, fingerprint configurations, and local storage generally cannot be transferred directly.

## Transferring cookies

1. In your current antidetect browser, export cookies for the account (usually in Netscape or JSON format)
2. In Dolphin Anty, create a new profile with matching settings
3. Open the 3-dot menu → **Import**
4. Paste or upload the exported cookie data
5. Start the profile and verify the session is still active

## Recreating profiles

For each account you're moving:

1. Note the User-Agent, proxy, and platform from the old profile
2. Create a new profile in Dolphin Anty
3. In the profile editor → Advanced tab, manually set the User-Agent to match
4. Assign the same proxy
5. Import cookies
6. Test the session

## Tips for a smooth migration

- Move accounts one at a time and test each before moving to the next
- Keep the old browser available temporarily as a fallback
- Warm up the profile in Dolphin Anty with a few natural browsing sessions before running paid activities
- If an account shows unusual login activity after the move, use the platform's account recovery flow before it gets flagged
