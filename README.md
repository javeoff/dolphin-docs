# Dolphin Anty Documentation

Dolphin Anty is an antidetect browser designed for multi-accounting, affiliate marketing, web scraping, and managing advertising campaigns across platforms like Facebook, Google, TikTok, and more.

## What is Dolphin Anty?

Dolphin Anty lets you create and manage multiple browser profiles, each with a unique fingerprint. Every profile appears as a distinct device to websites, preventing account linkage and detection.

**Key use cases:**
- Managing multiple advertising accounts on a single platform
- Affiliate marketing campaigns
- Web scraping and data collection
- E-commerce operations on multiple storefronts
- Social media management at scale

## How it works

Each browser profile in Dolphin Anty has its own isolated:
- Browser fingerprint (User-Agent, Canvas, WebGL, fonts, screen, audio, and 20+ other parameters)
- Cookie storage
- Local storage
- Proxy connection

All fingerprint spoofing happens at the browser engine level — no JavaScript injections that can be detected.

## Core features

| Feature | Description |
|---------|-------------|
| Browser profiles | Create unlimited isolated browser environments |
| Fingerprint management | Control 20+ fingerprint parameters per profile |
| Proxy integration | Assign HTTP/SOCKS4/SOCKS5 proxies to any profile |
| Team collaboration | Share profiles and assign roles to team members |
| Automation API | Integrate with Puppeteer, Playwright, and Selenium |
| Profile synchronizer | Mirror actions across multiple profiles simultaneously |
| Mass operations | Bulk start, stop, tag, transfer, and manage profiles |
| Cloud sync | Sync profiles, cookies, and extensions across devices |
